# Rock Paper Scissors 🎮

A simple Rock Paper Scissors game built with JavaScript.

* Play against the computer

* Score updates dynamically

* Leader’s score changes color

* Win/Lose messages displayed

* Endless gameplay

The computer makes random choices each round.
